 Presentizr - 1.0

Bootstrap-based theme (fully responsive), translation ready, integrated share buttons, integrated facebook comment plugin

== Description ==
Present your content using Presentizr. The theme ships with built-in tools like the facebook comments plugin. 
A new approach to menus includes to only show icons at first, and as soon as they are hovered the menu's titles appear.
Check it out in the demo!
